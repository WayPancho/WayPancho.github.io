相对论的数学基础

1. 指标记号

1.1 矢量

物理中的时空由四个坐标描述：

  • 时间：t=x0t = x^0t=x0

  • 空间：x=(x1,x2,x3)\boldsymbol{x} = (x^1,x^2,x^3)x=(x1,x2,x3)

将四个坐标写成xμx^\muxμ的形式，其中指标μ\muμ取0,1,2,30,1,2,30,1,2,3四个值，则其标示时空中的一点。

在时空中的xμx^\muxμ附近取一点，设其坐标为x^\mu + \dd{x}^{\mu}，则有

  • 位移： \dd{x}^\mu

我们认为位移是一个矢量，而四个分量\dd{x}^\mu则是这个四矢量的四个分量。

狭义相对论使用的Lorenz变换是对坐标的线性非齐次变换，对位移来说就是线性的齐次变换。在这个变换下

(\dd{x}^0)^2 - (\dd{x}^1)^2 - (\dd{x}^2)^2 - (\dd{x}^3)^2 \label{ds^2}

是一个不变量（采用自然单位制c=1c=1c=1）。

考虑由四个量构成的AμA^\muAμ，如果在坐标变换的同时AμA^\muAμ的四个量与\dd{x}^\mu变换的形式是相同的，则AμA^\muAμ称为逆变矢量（contra-variant vector）。

由于变换的一致性

(A^0)^2 - (A^1)^2 - (A^2)^2 - (A^3)^2 \equiv (A,A) \label{(A,A)}

也是一个不变量，称其为矢量长度的平方。

考虑两个逆变矢量AμA^\muAμ和BμB^\muBμ，则其标量积

A^0B^0 - A^1B^1 - A^2B^2 - A^3B^3 = (A,B) \label{(A,B)}

也是不变量。

为了方便地书写这些不变量，引入下标，定义

A_0 = A^0,\quad A_1 = -A^1,\quad A_2 = -A^2,\quad A_3 = -A^3 \label{lowering process}

于是\eqref{(A,A)}可以改写为

A0A0+A1A1+A2A2+A3A3A_0A^0 + A_1A^1 + A_2A^2 + A_3A^3 A0​A0+A1​A1+A2​A2+A3​A3

采用Einstein求和协定则可简记为AμAμA_\mu A^\muAμ​Aμ。采用同样的记号，不难验证\eqref{(A,B)}可以写成AμBμA_\mu B^\muAμ​Bμ或者AμBμA^\mu B_\muAμBμ​。

引入的四个量AμA_\muAμ​同样可以看成是某个矢量的分量，由于正负号的差别这种矢量在坐标变换下的规律与AμA^\muAμ是不一样的，这种新的矢量叫做协变矢量（covariant vector）。

1.2 张量

如果把两个逆变矢量AμA^\muAμ和BνB^\nuBν组合，则可以构成161616个量，记作AμBνA^\mu B^\nuAμBν，这十六个量构成了一个二阶张量（tensor of the second rank）。

当采取小写字母，如i,j,ki,j,ki,j,k作为指标时，往往标示三个分量，即字母遍取1,2,31,2,31,2,3；而采取希腊字母，如μ,ν\mu,\nuμ,ν等作为指标时，则往往标示四矢量，即字母遍取0,1,2,30,1,2,30,1,2,3。

由于构成张量AμBνA^\mu B^\nuAμBν的两个矢量是逆变的，对其分量之间的关系有约束，所有其具有一定的特殊性。如果把用这种方法组合出来的几个张量加起来，则可以得到一个更一般的二阶张量

Tμν=AμBν+A′μB′ν+A′′μB′′ν+⋯T^{\mu \nu} = A^\mu B^\nu + A^{\prime\mu} B^{\prime\nu} + A^{\prime\prime\mu} B^{\prime\prime\nu} + \cdots Tμν=AμBν+A′μB′ν+A′′μB′′ν+⋯

一般张量的重要特征是在坐标变换下其分量与AμBνA^\mu B^\nuAμBν具有相同的变换形式。

利用\eqref{lowering process}的降标处理，则可以降低TμνT^{\mu\nu}Tμν的某一个指标，得到TμνT_\mu{}^\nuTμ​ν或TμνT^\mu{}_\nuTμν​，也可以直接把两个指标都降低，得到TμνT_{\mu\nu}Tμν​。

在TμνT_\mu{}^\nuTμ​ν中令μ=ν\mu=\nuμ=ν，则得到TμμT_\mu{}^\muTμ​μ，按照Einstein求和协定，在一项中如果同一个指标出现两次则意味着求和，于是TμμT_\mu{}^\muTμ​μ是一个标量，同理TμμT^\mu{}_\muTμμ​也是一个标量，且按照降标处理的定义\eqref{lowering process}可以看出Tμμ=TμμT^\mu{}_\mu = T_\mu{}^\muTμμ​=Tμ​μ。

按照把指标不同的矢量组合在一起的办法，可以构成更高阶的张量。如果矢量全都是逆变的，则会得到一个指标都是上标的张量，然后通过降标处理可以得到具有任意数量上标和任意数量下标的一般张量。

如果令一个张量的一个下标等于某个上标，则意味着对这个指标求和，因此这个指标不再能标示分量，变成一个”哑巴的“指标，称其为哑指标（dummy index）。求和后张量少了两个能标示分量的指标，这样的过程叫做缩并（contraction）。例如，对于一个四阶张量TμνρσT^\mu{}_\nu{}_\rho{}^\sigmaTμν​ρ​σ，通过令σ=ρ\sigma = \rhoσ=ρ来缩并它，则得到的张量TμνρρT^\mu{}_\nu{}_\rho{}^\rhoTμν​ρ​ρ只有两个有效的指标μ\muμ和ν\nuν，因此是一个有十六个分量的二阶张量。如果通过令ν=μ\nu = \muν=μ来进一步缩并，则可以得到没有任何有效指标的量TμμρρT^\mu{}_\mu{}_\rho{}^\rhoTμμ​ρ​ρ，它只有一个分量，是一个标量。

由此可以看出，一个有效指标在一项中会且只会出现一次。如果一个指标出现了两次，则是一个意味着求和的哑指标，它可以被替换为任何一个这一项中没用过的其他希腊字母。一个指标在一项中不可能出现两次以上，否则是没有意义的。

2. 斜交轴

2.1 长度和标量积

考虑不正交的坐标轴，则需要将上面的一些定义拓展。

考虑\eqref{ds^2}这个不变量的形式的改变，这个特殊二次型将变成关于新坐标\dd{x}^\mu的另一个一般二次型

g_{\mu\nu} \dd{x}^\mu \dd{x}^\nu

按照第1节中的记号说明，上式是取遍μ\muμ和ν\nuν每一个值的求和。二次型的系数依赖于新坐标系，且由于上式中两个指标是可交换的，有限制条件gμν=gνμg_{\mu\nu} = g_{\nu\mu}gμν​=gνμ​，如果将gμνg_{\mu\nu}gμν​的取值排成四阶方阵(Mg)μν(M_g)_{\mu\nu}(Mg​)μν​，则限制条件表明MgT=MgM_g^T = M_gMgT​=Mg​，这样可以看出存在十个独立的系数。

按照逆变矢量的定义，逆变矢量AμA^\muAμ在斜交轴下的变换跟\dd{x}^\mu一样，所以\eqref{(A,A)}也将变成

gμνAμAνg_{\mu\nu}A_\mu A_\nu gμν​Aμ​Aν​

和第一节的逻辑一样，这是一个不变量，是矢量AμA^\muAμ长度的平方。

我们试图把(A,B)(A,B)(A,B)也迁移过来，考虑逆变矢量AμA^\muAμ和BμB^\muBμ，则Aμ+λBμA^\mu + \lambda B^\muAμ+λBμ也是一个逆变矢量，其长度的平方为

gμν(Aμ+λBμ)(Aμ+λBμ)=gμνAμAν+λ(gμνAμBν+gμνAνBμ)+λ2gμνBμBνg_{\mu\nu}(A^\mu + \lambda B^\mu)(A_\mu + \lambda B_\mu) = g_{\mu\nu}A^\mu A^\nu + \lambda(g_{\mu\nu}A^\mu B^\nu + g_{\mu\nu}A^\nu B^\mu) + \lambda^2 g_{\mu\nu}B^\mu B^\nu gμν​(Aμ+λBμ)(Aμ​+λBμ​)=gμν​AμAν+λ(gμν​AμBν+gμν​AνBμ)+λ2gμν​BμBν

由于它是一个不变量，关于λ\lambdaλ的零次项，一次项和二次项都必须不变。对于一次项，利用gμν=gνμg_{\mu\nu}=g_{\nu\mu}gμν​=gνμ​，可以得到

gμνAμBν+gμνAνBμ=2gμνAμBνg_{\mu\nu}A^\mu B^\nu + g_{\mu\nu}A^\nu B^\mu = 2g_{\mu\nu}A^\mu B^\nu gμν​AμBν+gμν​AνBμ=2gμν​AμBν

于是得到了gμνAμBνg_{\mu\nu}A^\mu B^\nugμν​AμBν是不变量，它对应着正交轴坐标系中的(A,B)(A,B)(A,B)，是二者的标量积。

对上面定义的矩阵MgM_gMg​，设它的行列式是ggg，将它叫做gμνg_{\mu\nu}gμν​的行列式。作为一个坐标变换，MgM_gMg​必须是满秩的，否则新坐标就不能在时空中提供四个独立的方向，从而不能作为描述时空的坐标轴。因此，可以得到结论g≠0g\neq 0g=0。对于正交轴来说，Mg=diag{1,−1,−1,−1}M_g = \mathrm{diag}\{1,-1,-1,-1\}Mg​=diag{1,−1,−1,−1}，于是g=−1g=-1g=−1。数学上，因为斜交轴可以通过对正交轴做一种连续的处理来得到，而且变化过程中ggg不能经过零，所以斜交轴中ggg仍然是负的。

2.2 协变矢量

在斜交轴坐标系里面，把协变矢量定义为

A_\mu = g_{\mu\nu} A^\nu \label{A_mu}

由于行列式ggg不等于000，可以求MgM_gMg​的逆矩阵，从而解出AνA^\nuAν，令结果为

A^{\nu} = g^{\mu\nu}A_\mu \label{g^munu}

这就是将gμνg^{\mu\nu}gμν设置为gμνg_{\mu\nu}gμν​的逆。按照伴随矩阵的理论，Mg−1=Mg∗/gM_g^{-1} = M_g^* / gMg−1​=Mg∗​/g，则每一个gμνg^{\mu\nu}gμν就对应着gμνg_{\mu\nu}gμν​的一个代数余子式除以行列式，可以通过行列式性质给出gμν=gνμg^{\mu\nu} = g^{\nu\mu}gμν=gνμ。

按照上面的假设，有（把\eqref{g^munu}代入\eqref{A_mu}里，且为了不让同一项里出现三次μ\muμ，用ρ\rhoρ代替\eqref{g^munu}里的哑指标）

Aμ=gμνgνρAρA_\mu = g_{\mu\nu}g^{\nu\rho}A_\rho Aμ​=gμν​gνρAρ​

这个式子对AμA_\muAμ​的四个分量都成立，从而

gμνgνρ={1,μ=ρ0,μ≠ρg_{\mu\nu}g^{\nu\rho} = \begin{cases} 1,\quad \mu=\rho\\ 0,\quad \mu\neq \rho \end{cases} gμν​gνρ={1,μ=ρ0,μ=ρ​

将其记为gμρ=gμνgνρg_\mu^{\rho} = g_{\mu\nu}g^{\nu\rho}gμρ​=gμν​gνρ，由于gμνgνρ=gμνgνρg_{\mu\nu}g^{\nu\rho} = g^{\mu\nu}g_{\nu\rho}gμν​gνρ=gμνgνρ​，其中包含了gμρg_\mu{}^\rhogμ​ρ和gμρg^{\mu}{}_{\rho}gμρ​两种可能性，不区分这两者。由上面的结果看出，gμρg_\mu^{\rho}gμρ​的作用就是用ρ\rhoρ替换μ\muμ或用μ\muμ替换ρ\rhoρ，即

gμρAμ=Aρg_{\mu}^{\rho} A^\mu = A^{\rho} gμρ​Aμ=Aρ

和

gμρAρ=Aμg_{\mu}^{\rho}A_{\rho} = A_\mu gμρ​Aρ​=Aμ​

总结上面的式子，公式\eqref{A_mu}说明gμνg_{\mu\nu}gμν​可以用来降低上标，公式\eqref{g^munu}说明gμνg^{\mu\nu}gμν可以用来升高下标。如果一个指标被升高后又被重新降低，gμρg_{\mu}^{\rho}gμρ​说明结果将会回归到变换之前。由于张量是由矢量组合而来的，这些升降指标的功能显然对张量也管用。

特殊地，考虑升高gμνg_{\mu\nu}gμν​的指标，则有

gνα=gαμgμνg_{\nu}^{\alpha} = g^{\alpha \mu}g_{\mu \nu} gνα​=gαμgμν​

这和前文是一致的。如果考虑gμνg_{\mu\nu}gμν​的对称性，可以把上式左边写成gανg^{\alpha}{}_{\nu}gαν​，这样与右端的一致性就更强了。

进一步升高指标ν\nuν，可以得到

gαβ=gνβgναg^{\alpha\beta} = g^{\nu \beta} g_{\nu}^{\alpha} gαβ=gνβgνα​

总的而言，gμν,gνμ,gμνg_{\mu\nu},g_{\nu}^{\mu},g^{\mu\nu}gμν​,gνμ​,gμν三者可以用来对指标进行升降或替换，升降指标的规则对这三者本身也是适用的。
